# 量價結構判讀框架

## 四種基本型態

### 1. 價漲量增（健康上攻）
- 意義：多方主導，資金積極進場
- 操作含義：趨勢延續，可順勢操作
- 注意：若量能過度放大（爆天量），留意短線過熱

### 2. 價漲量縮（背離警訊）
- 意義：追價意願不足，漲勢缺乏資金支撐
- 操作含義：謹慎追高，留意假突破
- 注意：若連續 2 日以上量縮創高，背離風險升高

### 3. 價跌量增（恐慌賣壓）
- 意義：空方主導，多方信心瓦解
- 操作含義：避免抄底，等待量縮止跌訊號
- 注意：若跌破關鍵支撐（5日/10日/月線），下檔空間擴大

### 4. 價跌量縮（自然回檔）
- 意義：賣壓有限，屬正常技術性修正
- 操作含義：回檔找支撐，可留意低接機會
- 注意：若量縮至 5 日均量 50% 以下，可能接近止跌

## 族群強弱判別

### 真強 vs 誘多
- **真強**：早盤開高走高 → 午盤維持強勢 → 尾盤收最高或接近最高
  - 特徵：量增、法人買超、多檔族群同步走強
- **誘多**：早盤跳空開高 → 盤中逐步走低 → 尾盤翻黑或收長上影線
  - 特徵：量大但收黑、法人趁拉高出貨、族群內個股分化

### 真空 vs 誘空
- **真空**：開低走低全日弱勢，量增跌破支撐
- **誘空**：早盤急殺 → 盤中止跌 → 尾盤拉回，收下影線
  - 特徵：急殺後量能萎縮、法人逆勢買超

## 外資期現貨組合判讀

| 現貨 | 期貨 | 組合訊號 | 操作含義 |
|------|------|----------|----------|
| 買超 | 多單增 | 真多 | 方向看好，順勢偏多 |
| 賣超 | 空單增 | 真空 | 方向看壞，防禦為主 |
| 買超 | 空單增 | 避險買進 | 謹慎偏多，留意反轉 |
| 賣超 | 多單增 | 誘空訊號 | 底部可能浮現，關注反彈 |

## 關鍵價位層級

1. **短線**：5 日均線、10 日均線
2. **中線**：月線（20 日均）、季線（60 日均）
3. **長線**：半年線（120 日均）、年線（240 日均）
4. **心理關卡**：整數關（如 40,000、41,000）

站上 = 收盤價 > 均線且隔日未跌破
跌破 = 收盤價 < 均線且量增確認

## 使用原則

- 判讀時必須結合「型態 + 量能 + 法人動向」三者
- 單一指標不足以下結論，需交叉驗證
- 對判讀結果要明確表態（偏多/偏空/中性），不要模棱兩可
- 提供具體價位（支撐/壓力），不要只說「留意」
